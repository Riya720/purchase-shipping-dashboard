# 📊 Purchase & Shipping Analysis Dashboard (Excel)

## 📌 Project Overview

This project presents an interactive **Purchase & Shipping Analysis Dashboard** built using Microsoft Excel. It provides deep insights into purchasing behavior, shipping patterns, and product performance, helping businesses make data-driven decisions.

The dashboard transforms raw transactional data into meaningful visual insights using Excel’s advanced features.

---

## 🎯 Objectives

* Analyze total purchase value and item distribution
* Track year-on-year performance trends
* Understand customer diversity and buying behavior
* Monitor product category performance
* Evaluate purchase frequency and seasonal trends

---

## 📊 Key Metrics

* **Total Purchase:** $578,951
* **Total Items Sold:** 10,745
* Category-wise analysis:

  * Accessories
  * Clothing
  * Footwear
  * Outerwear

---

## 📈 Dashboard Features

### 🔹 Purchase Analysis

* Product-wise purchase breakdown
* Monthly purchase trends
* Seasonal purchase distribution

### 🔹 Performance Tracking

* Year-on-Year variance comparison (2023 vs 2024)
* Growth trend visualization

### 🔹 Customer Insights

* Gender diversity distribution
* Purchase behavior by size (S, M, L, XL)

### 🔹 Operational Insights

* Purchase frequency (Weekly, Monthly, Quarterly, etc.)
* Payment method distribution
* Quantity purchased by color
