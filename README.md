# Purchase & Shipping Dashboard (Excel)

This project is an Excel dashboard created to analyze purchase and shipping performance.

Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

Dashboard Insights
- Total purchases
- Shipping performance
- Category wise purchase analysis
- Monthly trends

Purpose
To demonstrate Excel data analysis and dashboard creation skills.
